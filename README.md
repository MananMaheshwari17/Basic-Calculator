# 🧮 Calculator Web App

A simple, responsive, and stylish calculator built using **HTML**, **CSS**, and **JavaScript**. It supports all basic arithmetic operations and has a clean user interface.


## 🚀 Features

- Basic operations: `+`, `-`, `*`, `/`, `%`
- Clear (`AC`) and Backspace (`BS`) buttons
